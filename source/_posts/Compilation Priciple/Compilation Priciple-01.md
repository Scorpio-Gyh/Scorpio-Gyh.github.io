# 一个简单编译器实例

* 源语言：加法表达式Sum

  * 两种语法形式：
    * 整形数字 n
    * 加法 e1 + e2
* 栈式计算机(例如jvm)

  * 一个操作数栈
  * 两条指令：
    * 压栈指令 push n
    * 加法指令 add
* 任务：编译程序1 + 2 + 3到栈式计算机
